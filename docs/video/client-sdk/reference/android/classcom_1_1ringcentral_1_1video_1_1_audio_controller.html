Detailed Description

The AudioController class is a managing class to control the audio-related functions and states in an active meeting session, such as mute or unmute local or remote audio.

◆ disableAudio()

abstract long com.ringcentral.video.AudioController.disableAudio ( )
abstract

Disables the audio session.

A successful call of disableAudio triggers the onLocalAudioStreamStateChanged event callback to the local user. For the local track, the client SDK will stop capturing frames from the audio source and not sending any audio data to the server. For the remote track, the server will not forward any audio streams to the local user.

Returns
0 means the action succeeds or fails otherwise

The RingCentral video client SDK uses the AudioEventHandler interface class to send audio-related event callbacks to the application, and the application inherits the methods of this interface class to retrieve those event callbacks.
